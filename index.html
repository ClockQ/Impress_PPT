三：Vuex介绍

Vuex是一个专为Vue.js应用程序开发的状态管理模式。它采用集中式存储管理应用的所有组件的状态，并以相应的规则保证状态以一种可预测的方式发生变化。

解决的问题：

多个视图依赖于同一状态。

来自不同视图的行为需要变更同一状态。

应用

每一个Vuex应用的核心就是store,"store"就是一个容器。注意：不能直接改变 store 中的状态。改变 store 中的状态的唯一途径就是显式地提交 (commit)

1. state

2. getter

3. mutation (同步)

4. action (异步)

5. module
